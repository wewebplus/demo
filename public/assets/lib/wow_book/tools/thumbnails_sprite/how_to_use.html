About Wowbook thumbnails sprite generator

This is a simple script that helps you to create thumbnails sprite for wowbook. For more information about "thumbnails sprite" read the section "Thumbnails" in the plugin documentation.

Limitations

This tool will work only with "image only" books, ie, books where each page is a img tag that fills the whole page. Works with image only books created manually or using the attribute "data-image" or the configuration option "images".

This tool was tested only with the most recent version of Firefox and Chrome.

How to use

Include the script "thumbnails_generator.js" in the webpage that contains the book. Because thumbnails_generator uses jQuery, the script tag must be placed after the tag that includes jQuery. 

	<script src="thumbnails_generator.js"></script>

To determine the size of the generated thumbnails, the tool will use the configuration options "thumbnailWidth" and "thumbnailHeight". You can also use the option "thumbnailScale".

Open the webpage in your browser. On the top left corner of the page, you're gonna see a white box with a button "Generate thumbnails". Click it.

Wait for the tool to load all the images in the book. When all the pages are loaded click on the link "Click to download thumbnails". Generate the thumbnails can take some time depending on the how much images the tool has to process.

After all the images are processed and the thumbnails sprite image is done, the browser will ask where do you want to save the image file. Save it on your webpage directory.

Set the configuration option "thumbnailsSprite" to point the image you just downloaded. 

	$('#book-element-id').wowBook({
	   height : 500,
	   width  : 600
	   // other configuration options
	   thumbnailsSprite: "path/to/the/image/file.png"
	});

Remove the script "thumbnails_generator.js" from your page.
